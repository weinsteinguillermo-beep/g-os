# Proyecto: Outdoor Import

## Tipo

Proyecto comercial / importaciones.

## Contexto

Outdoor Import debe documentarse como linea comercial o proyecto con foco en productos, proveedores, margenes, demanda y canales.

## Objetivo

Evaluar y ordenar oportunidades de importacion con criterio comercial.

## Prioridad Inicial

Media.

## Proximas Acciones

- Definir alcance.
- Identificar proveedores.
- Registrar productos objetivo.
- Evaluar margen, demanda y riesgo logistico.
