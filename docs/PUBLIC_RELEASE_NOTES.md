# Public Release Notes

## G-OS v0.1

Fecha de lanzamiento preparada: 25 de junio de 2026.

## Que Es

G-OS v0.1 es la primera version publica del sistema operativo ejecutivo de Guillermo Weinstein.

No es un sitio web. Es el primer panel ejecutivo movil para validar una rutina real de decision, captura de ideas, cierre del dia y preparacion de trabajo para Codex.

## Incluye

- Daily Briefing.
- Modo manana.
- Modo cierre.
- Captura de ideas.
- Edicion simple de ideas.
- Decisiones con estado y prioridad.
- Proyectos clave.
- Generacion de mision Codex.
- Copiar mision Codex.
- Cierre del dia copiable.
- Aprendizaje del dia.
- Exportar e importar JSON.
- PWA minima con manifest e icono.

## No Incluye Todavia

- Backend.
- APIs.
- Login.
- Sincronizacion entre dispositivos.
- Integraciones con Gmail, Drive, Airtable, n8n o Power BI.
- Funcionalidad offline completa.

## Filosofia De La Version

Primero validar uso diario.

Despues crecer.

## Riesgos Conocidos

- Los datos viven en `localStorage` del navegador.
- No hay sincronizacion.
- La instalacion en iPhone depende de Safari.
- GitHub Pages puede tardar unos minutos en reflejar cambios.

## Proximo Hito

Definir la primera fuente de verdad real: JSON manual, Google Sheet o Airtable.
