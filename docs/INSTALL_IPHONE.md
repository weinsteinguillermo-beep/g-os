# Instalar G-OS En iPhone

## Objetivo

Agregar G-OS a la pantalla de inicio del iPhone para que se sienta como una aplicacion.

## Requisitos

- G-OS publicado en una URL publica.
- Abrir la URL desde Safari en iPhone.

## Pasos

1. Abrir Safari en iPhone.
2. Entrar a la URL publica de G-OS.
3. Tocar el boton de compartir.
4. Elegir `Agregar a pantalla de inicio`.
5. Confirmar nombre: `G-OS`.
6. Tocar `Agregar`.

## Uso Diario

Desde la pantalla de inicio:

1. Abrir G-OS.
2. Tocar `Modo manana`.
3. Revisar briefing, decisiones y proyectos clave.
4. Capturar ideas si aparecen.
5. Al final del dia, tocar `Modo cierre`.
6. Guardar aprendizaje y copiar el cierre si corresponde.

## Nota

La version v0.1 guarda datos en el navegador del dispositivo. Si se usa G-OS en otro dispositivo, los datos locales no se sincronizan todavia.

Para respaldo manual, usar `Exportar datos`.
